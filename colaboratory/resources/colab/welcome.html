Welcome to coLaboratory
About
coLaboratory is an interactive front end for the Jupyter/ IPython Notebook, offering full integration with Google Drive.
Like the IPython Notebook, coLaboratory lets you create documents that combine live code with narrative text, but coLaboratory allows editing of documents by multiple people at once, and stores them on Google Drive.
The coLaboratory website allows you to view and edit notebooks. It also allows you to connect to an IPython kernel running on your own system, so that you can execute code cells.
Step 1: Install a IPython
First, install IPython 2.2 on your local machine. Note that no other version of IPython will work with coLaboratory.
Step 2: Configure IPython
Download and run **this script** to configure IPython and generate a self-signed SSL certificate. This enables a secure connection between your local kernel and our web server.
Step 3: Launch IPython
Next start IPython on your local machine using the profile you created: 
 ipython notebook --profile=colab
Replace colab with the IPython profile you are using.
Step 4: Install coLaboratory
Please click on the button below to install coLab in Google Drive.
Congrats, you already have the coLaboratory Drive application installed.
Step 5: Open a Notebook
You can click on the buttons below to either create a new notebook or open an existing notebook from Google Drive.
